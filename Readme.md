# Fb Test App

A basic Rails Omniauth project with MongoDB.

## Setup
    git clone git@github.com:xta/Fb_Mongoid_Test.git
    cd Fb_Mongoid_Test
    bundle
    mv config/config.yml config/config.yml.example

place Facebook API keys in `config/config.yml`

    rails s

## Stack
* Ruby on Rails
* MongoDB
